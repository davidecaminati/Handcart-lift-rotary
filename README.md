# Handcart-lift-rotary
HandCart that can lift and rotate

# Requisiti
resistente fino almeno 60Kg (30Kg ripiano basso + 30Kg ripiano alto)
deve sterzare in 123 cm (ascensore)
il piano alto deve poter ruotare ed essere bloccato in una posizione
quando è tutto basso, il piano superiore deve essere a 100 cm da terra (chiedere conferma)
quando è tutto alto, il piano superiore deve essere a 120 cm da terra (chiedere conferma)
il piano superiore non deve essere sganciabile (nemmeno sollevando il dispositivo)
l'abbassamento del dispositivo non deve mettere in pericolo il materiale che è appoggiato al piano inferiore
